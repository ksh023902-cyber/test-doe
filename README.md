# test-doe

Short-form(한 줄) 커뮤니티 웹앱 (Vite + Vanilla JS).

## Run

```sh
npm install
```

```sh
npm run dev
```

## Features

- Home shorts feed (scroll-snap)
- Click to open detail page
- Create posts / create shorts (localStorage)
- Like / save / share / report
- Report-based auto hiding
- Recommendation mix: light “썰” 50% + deep 50%
